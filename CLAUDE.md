# 책인감 프론트엔드

독서 전용 SNS 서비스 **책인감**의 Next.js 프론트엔드입니다.

---

## 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 프로젝트명 | 책인감 (책 + 인감 — 책에 남기는 나만의 도장) |
| 서비스 유형 | 독서 전용 SNS |
| 백엔드 API | `http://localhost:8080` |
| 스택 | Next.js · TypeScript · Tailwind CSS |

---

## 주요 화면

- **피드** — 팔로우한 사람들의 독후감 타임라인
- **로그인 / 회원가입** — JWT 기반 인증
- **독후감 작성** — 책을 선택하고 감상을 기록
- **책 검색** — 책 제목·저자 검색
- **프로필** — 사용자 정보 및 활동 요약
- **서재** — 읽은 책 / 읽고 싶은 책 목록

---

## API 인증

모든 인증이 필요한 요청에는 JWT 토큰을 헤더에 포함합니다.

```
Authorization: Bearer <token>
```

토큰은 로그인 후 응답에서 받아 `localStorage` 또는 `httpOnly` 쿠키에 저장합니다.

---

## 디자인 컨셉

- **책 감성** — 종이, 잉크, 책장을 연상시키는 따뜻하고 차분한 분위기
- **톤** — 깔끔하고 따뜻한 크림·베이지·브라운 계열
- **폰트** — 가독성 좋은 명조/고딕 혼합 (책 느낌 강조)
- 과도한 애니메이션보다 **여백과 타이포그래피**로 완성도를 높입니다
